QUIZ SYSTEM — HOW IT WORKS
🚀
The quiz system operates in 3 stages: Info (enter name & email), Quiz Taking (MCQ with 30-sec timer, auto-advance on timeout), and Results (score summary with percentage bar). Questions fetched from Supabase /questions table by display_order.
→ Enter name & email to start.
Scores are saved to /scores table. Confetti fires on every correct answer. Timer shows visual warning when under 10 seconds. Buttons highlight green on correct, red on wrong.
→ Continue to leaderboard...
STAGE 1 — INFO
Name (required), Email (optional). Clean form with focus states, input validation, and "Start Quiz" CTA.
STAGE 2 — QUIZ
Options A, B, C, D · Progress bar · 30s countdown · Next Question button enabled after answering.
STAGE 3 — RESULTS
Final score (e.g. 8/10) · Percentage bar · Dynamic performance message · Replay or Home buttons.
